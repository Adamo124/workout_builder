# Workout Builder

Workout Builder är ett Pythonprogram som skapar ett träningspass baserat på vald muskelgrupp.
Programmet läser övningar från en fil, bearbetar datan och skriver ut ett färdigt träningspass
till användaren.

## Installation

Klona repot och skapa en virtuell miljö:

```bash
git clone <repo-url>
cd workout_builder
python -m venv venv
source venv/Scripts/activate  # Windows
pip install -r requirements.txt
